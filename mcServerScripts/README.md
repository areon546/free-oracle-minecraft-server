## secrets.sh template

```sh
#! /bin/bash

DRAGON_PRIVATE_PATH="path"
DRAGON_USERNAME="user"
DRAGON_IP="xxx.xxx.xxx.xxx"
DRAGON_API_KEY="API_KEY"


export DRAGON_IP
export DRAGON_USERNAME
export DRAGON_PRIVATE_PATH
export DRAGON_API_KEY
```
